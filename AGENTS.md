# AGENTS.md

Operating rules for AI coding agents working in this repository.

## Repository Memory

- Read `docs/PROJECT_MEMORY.md` before every milestone.
- Follow `docs/ARCHITECTURE_EXECUTION_PLAN.md`.
- Execute one milestone only per task.
- Do not jump ahead.

## Product Boundaries

- Do not add Shopify integration before the planned milestone.
- Do not add database, LLM, or API infrastructure early.
- Do not change product positioning.
- This is not an SEO scanner.
- This is a Shopify-native AI Representation Optimizer.

## Verification

- Always run `npm run typecheck`, `npm run lint`, and `npm run build` after changes.
- Always report files changed and commands run.

## Git Identity

- Always commit successful milestone changes using Sraman Nandi's git identity.
- Never commit as claude, codex, or any bot.
